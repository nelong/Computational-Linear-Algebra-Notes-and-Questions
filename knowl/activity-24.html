Activity 1.7.2.

  1. How many vectors are in \(Span(\{ \langle 1,1 \rangle ,\langle 1,-1 \rangle \})\text{?}\)
  2. Is there any vector in \(\mathbb{R}^2\) that is not in \(Span(\{ \langle 1,1 \rangle ,\langle 1,-1 \rangle \})\text{?}\)
  3. How many vectors are in \(Span(\{ \langle 1,1,1 \rangle ,\langle 1,-1,1 \rangle \})\text{?}\)
  4. Is there any vector in \(\mathbb{R}^3\) that is not in \(Span(\{ \langle 1,1,1 \rangle ,\langle 1,-1,1 \rangle \})\text{?}\)
  5. Try to write out the set of vectors in \(Span(\{ \langle 1,1,1 \rangle ,\langle 1,-1,1 \rangle \})\text{?}\) Hint: write the corresponding system of equations, then use the solution set of this system to write out the exact vector form of \(Span(\{ \langle 1,1,1 \rangle ,\langle 1,-1,1 \rangle \})\text{.}\)
  6. Is there any vector in \(\mathbb{R}^2\) that is not in \(Span(\{ \langle 1, 0\rangle ,\langle 0,1 \rangle, \langle 1,1 \rangle \})\text{?}\)
