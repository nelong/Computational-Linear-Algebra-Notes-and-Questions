Solution 2.6.8.1.

We prove this by contradiction. Suppose that there is an eigenspace with dimension \(0\text{.}\) Then \(E_\lambda = \{\vec{0}\}\text{,}\) since this is the only vector space of dimension \(0\text{.}\) However, we now see that \(E_\lambda\) does not contain any nonzero vectors, and thus cannot contain any eigenvectors and therefore is not an eigenspace.
