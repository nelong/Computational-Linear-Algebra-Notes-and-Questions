The following sets are also vector spaces:
  1. The set of all polynomials (in variable \(t\)) denoted \(\mathbb{P}\text{.}\)
  2. \(F(S,\mathbb{R})\text{,}\) the set of functions from a set \(S\) to the real numbers. We will take a closer look at this vector space in the next problem.
  3. \(\{\vec{0}\}\text{,}\) the trivial vector space.
